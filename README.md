SARAH-Plugin-Milight
===========================

Le module "Milight" permet a Sarah de piloter des ampoules led Wifi (Blanches ou RGBW) via une passerelle Wifi.

Très simple d’utilisation, il suffit de brancher la "passerelle" en USB pour l’alimenter, de parametrer l'acces a votre réseau Wifi et voilà vous contrôler vos ampoules avec Sarah.
